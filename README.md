# LMS

---

### LMS기능을 간단하게 구현한 프로젝트

### 구현된 기능

- 회원가입 & 로그인
- 이메일 인증 및 활성화
- 회원 관리 구현 (관리자 전용)
- 강의 관리 구현 (관리자 전용)
- 강의 수강 관리 (관리자 전용)
- 배너 관리 (관리자 전용)
- 배너 표시 
- 강의 신청

### 사용된 도구 및 의존성

- IntelliJ Community (IDE)
- Maven (build)
- Spring Boot (framework)
- Spring Security (authentication)
- ThymeLeaf (template engine)
- MariaDB (database)
- JPA (ORM)
- MyBatis (SQL Mapper)
- Spring Boot Starter Mail
- Lombok
